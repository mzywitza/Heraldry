Heraldry Notes
==============

This is the source release of Heraldry. If you are an end-user of the application, please download a binary release instead.

Folder Structure
----------------

- build: Build results, binaries etc.
- doc: Documentation
- lib: Required libraries
- src: Source code
- tools: Build scripts and tools

Building Heraldry
-----------------

Heraldry uses PowerShell for building. To build the project yourself, you need PowerShell 2.0 installed on the PC.

Solution Structure
------------------

All projects are contained in a single solution.

- Heraldry.Storage: Database access and models (ORM)
- Heraldry.Storage.Tests: Unit tests
